# Discovery Evidence

Store mandatory search evidence here before implementation.

- Required log file: `discovery-log.md`
- Template: `../tpl/discovery-log-tpl.md`
- Each entry must include:
  - `Source/来源`
  - `Checked/查看内容`
  - `Relevance/关联度`
  - `Usefulness/有用程度`
  - `Value/价值`
  - `Reference Plan/参考计划`
